参赛形式与内容
形式

1. 初赛通过网络平台提交参赛作品，复赛和决赛为现场进行。

2. 参赛作品的内容不限，只要能够充分展示出参赛者在艺术、科技等方面的优秀技能即可。

内容

1. 艺术才能展示：包括舞蹈、器乐、歌唱、戏剧、表演、书法、绘画、雕塑、武术、曲艺等

2. 科技才能展示：包括各种自主发明创造作品

3. 其它才能展示：英语演讲、中文演讲等

比赛时间

1. 在线报名：4月8日至5月21日，在线提交基本报名信息，完成参赛报名。

2. 初赛阶段：4月8日至5月21日，在线提交参赛作品。5月26日前公布晋级复赛人选。

3. 复赛阶段：5月28日至6月30日，现场参与选拔。7月18日前公布复赛的最终结果。

4. 决赛阶段：8月18日至8月27日，现场与选拔。9月12日前公布决赛的最终结果。

报名办法及参赛要求
参赛资格确认

1. 参赛选手登录跟谁学官方活动页面，注册为跟谁学学生用户，在线填写报名信息后提交报名。

2. 报名资格获得确认后，可当即在线提交初赛的视频作品。

初赛

1. 获得参赛资格者在初赛规定的时间内，在官方活动页面成功提交参赛作品，视为正式参加了参赛。每名参赛者只能提交1件参赛作品。

2. 初赛提交作品为时长不超过10分钟的视频，视频格式可以是AVI、WMA、RMVB、MP4、FLASH等常见视频格式，推荐格式为MP4。

3. 如果初赛作品不便于在录制视频的同时进行创作（如书法、绘画、雕塑及科技发明等），则在视频中展示作品成品的同时，需由活动参赛者本人对作品进行讲解。

4. 初赛作品要求为原创或参赛者拥有合法的版权。否则，将会被取消参赛资格；如因视频与作品版权问题而产生纠纷，则参赛者或其法定监护人负责。

5. 初赛作品提交成功，即进入网络评选环节。

6. 初赛环节得票总数位列所在年级组别（如高中组）前50%的参赛者，将进入专家评审阶段。专家评审将依据参赛作品的类别，进行综合评选，评选出入围复赛的人员。

7. 初赛评审费10元/人，在跟谁学平台直接在线支付。

复赛

1. 复赛为现场评选，大赛组委会将组织评委在全国50个城市的复赛现场巡回评审，分别按年级组别和参赛作品类别进行评选。

2. 参加复赛的选手在复赛现场创作或展示与初赛一致或相似的作品，评委进行现场打分。每位通过初赛的选手只能参加一场复赛。

3. 获得复赛资格者，均可获得大赛组委会的认证培训机构提供的价值300元的免费复赛指导服务。认证培训机构名录可在活动官方网站查询。

4. 复赛评审费20元/人，可在跟谁学平台直接在线支付，亦可线下支付，复赛的各地承办方可以根据当地组织工作实际情况，另行收取诸如培训辅导等类费用，参赛人员如有疑问，可向大赛组委会咨询。

5. 选手因参加复赛而发生的交通食宿等费用自理。

决赛

1. 决赛为现场评选，大赛组委会将组织评委在北京、上海等10个区域中心城市进行现场评审，分别按年级组别和参赛作品类别进行评选。决赛评委将由各个领域的权威专家和社会知名人士担任。

2. 获得决赛资格者在决赛现场创作或展示与初赛、复赛一致或相似的作品，评委进行现场打分，每位通过复赛的选手只能参加一场决赛。

3. 获得决赛资格者，均可获得大赛组委会的认证培训机构提供的价值600元的免费复赛指导服务。认证培训机构名录可在活动官方网站查询。

4. 决赛不收取任何参赛费用，但选手因参加决赛而发生的交通食宿等费用自行承担。

评审组织与评选办法
初赛

初赛评审包括网络评审和专家评审两个环节。

1. 网络评选自4月8日开始，所有跟谁学注册用户均可以为参赛者投票。每个注册用户每天只能为一位参赛者投1票，但可以为多个参赛者投票。网络评选将于5月7日18:00结束，在此之前的网络得票记为有效得票。

2. 专家评审由大赛组委会组织各相关领域的优秀教师进行评选。

复赛

复赛评委由大赛组委会制定统一评审标准，在复赛现场进行评审，每位复赛选手将接受2位评委评审。

决赛

决赛分成现场评审和复核评审两个环节。

1. 大赛组委会组织部分评委到决赛现场进行评审，每位决赛选手将接受3位评委现场评审。

2. 通过现场评审的选手信息将交由5位评委进行复核评审，并最终评选大赛获胜者。

赛事奖励

1. 通过决赛且综合排名前20%者（最多100名），可获得“中英文化交流大使”称号及“中英文化交流杰出奖”证书，奖励如下：

1) 获得价值40000元的“2017年寒假英国游学营”的机会，造访牛津大学、剑桥大学、白金汉宫、大英博物馆等，参与英国政府或学校主办的文化主题活动，游学归来后，通过跟谁学平台向国内学生分享出访经历和取得的成果；

2) 大学生获奖者获得优先担任跟谁学提供的诸如国际游学营助教机会，以助教身份带领低年级学生出访各国；

3) 跟谁学提供的价值1500元的外语培训课程；

4) 大学生获奖者，优先获得跟谁学提供的全职、实习岗位机会；

5) 高中生和大学生获奖者，优先获得跟谁学创始人陈向东（中国人民大学博士、哈佛商学院校友、全国青联委员）亲笔签名的留学、就业、创业推荐信。

2. 通过决赛且综合排名后80%者（最多400名），可获得“中英文化交流优秀奖”证书，奖励如下：

1) 获得价值3000元的“2017年寒假北京中英文化交流冬令营”的机会，与国外学生和教师代表举行文艺汇演或文化交流活动，游学归来后，通过跟谁学平台向国内学生分享经历和取得的成果；

2) 跟谁学提供的价值1500元的外语培训课程；

3) 大学生获奖者，优先获得跟谁学提供的全职、实习岗位机会。

3. 成功进入决赛者，获得大赛组委会认证培训机构提供的最高价值600元的赛前辅导课程及大赛组委会颁发的”中英文化交流决赛入围奖“证书；

4. 成功进入复赛者，获得大赛组委会认证培训机构提供的最高价值600元的赛前辅导课程。

报名参赛办法
公开报名

可以登录跟谁学网站（www.genshuixue.com），通过大赛专题页面直接报名并提交参赛作品。

1. 参赛者报名成功后，可推荐他人报名，达到一定条件者，可获得相应奖励。

2. 一名参赛者推荐参赛人员达20人，且有1名被推荐人在决赛环节胜出者，推荐人将获得价值3000元的“2017中外文化交流冬令营”名额1个，且该名额不限本人使用。

培训机构或老师渠道

各类培训机构和老师均可以组织学生统一报名参赛，并根据组织情况获得相应奖励，具体组织方法可向跟谁学各地分公司、合作运营中心或总部客户服务中心咨询。

全日制学校渠道

各类学校可以组织本校学生统一报名参赛，并根据组织情况获得相应奖励，具体组织方法可向跟谁学各地分公司、合作运营中心或总部客户服务中心咨询。

“邀请好友参赛”活动

1. 直接通过活动官网报名成功后，参赛者再推荐一人成功报名者，本人复赛评审费免除。被推荐人在报名时输入推荐人的编号作为参赛编号并且报名成功，则视为成功推荐。

2. 一名参赛者推荐参赛人员达20人，且有1名被推荐人通过决赛者，将获得价值3000元的“2017北京中英文化交流冬令营”名额1个，该名额不限本人使用。
