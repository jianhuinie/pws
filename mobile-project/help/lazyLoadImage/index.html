LazyLoadImage Demo List

  1. 方式一:

    外层容器定宽高(或固定宽高比), 图片裁剪, 铺满容器

    用法:外层div定宽高或用padding-bottom固定宽高比, 可以设置默认背景图, 里面的img标签absolute定位, 宽高都是100%

    demo

  2. 方式二:

    外层容器指定宽高, 图片等比缩放, 居中展示

    demo

  3. 方式三:

    不定宽高, 图片最大宽度100%, 通常用于图文详情

    demo

  4. 方式四:

    控制局部图片懒加载,例如tab切换时

    使用方式:

    1. 如果没有新tab的内容没有变动, 激活新tab时重新调用lazyLoadImage.refresh()即可

    2. 如果tab内容为动态创建, 插入dom节点后需要调用lazyLoadImage.init传入局部容器重新查找需要懒加载的图片

    demo

  5. 方式五:

    在IScroll中实现图片懒加载

    使用说明: 局部滚动容器增加data-scroll-wrap属性, 并在滚动时触发lazyLoadImage.refresh()

    demo
